# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.1.x   | :white_check_mark: |
| 3.0.x   | :x:                |
| < 3.0   | :x:                |

## Reporting a Vulnerability

To report a security vulnerability, please email [branislavlang@gmail.com](mailto:branislavlang@gmail.com). We will respond within 48 hours.

**Please do not open a public issue for security vulnerabilities.**
